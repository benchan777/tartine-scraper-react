# How to run

## Clone the repo
- `git clone https://github.com/benchan777/tartine-scraper-react.git`

## Setting up and running the backend
1. Set up base files

   ```
   $ cd backend
   $ npm install
   ```

1. Start the backend

   `npm start`

## Setting up and running the frontend
1. Set up base files

    ```
    $ cd frontend
    $ npm install
    ```

1. Start the frontend

    `npm start`
